# Pacote para Codex - Redes ISP Academy

Este pacote contém a apostila convertida para Markdown/texto e imagens de referência de layout para enviar ao Codex.

## Arquivos principais

- `content/apostila-redes-isp.md` - arquivo recomendado para o Codex usar como fonte principal.
- `content/apostila-redes-isp.txt` - versão texto alternativa.
- `references/tela-trilha-modulo.png` - referência visual da tela de trilha/módulo.
- `references/tela-continuar-aula.png` - referência visual da tela de aula.
- `CODEX_INSTRUCTIONS.md` - instruções resumidas para o Codex.

## Recomendação
Coloque esta pasta ou o conteúdo deste ZIP dentro do repositório do projeto e peça ao Codex para usar `content/apostila-redes-isp.md` como fonte principal.
